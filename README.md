# 🩺 Assistente de Medicamentos (CLI)

**Versão:** 1.0.0

## 🎯 O Problema (Dor Real)
Cuidadores de idosos e familiares frequentemente se perdem nos horários, dosagens e restrições de múltiplos medicamentos. Erros de dosagem ou esquecimentos podem gerar consequências graves para a saúde do paciente.

## 💡 A Solução
Uma aplicação de linha de comando (CLI) simples, robusta e interativa que permite o cadastro de pacientes e a gestão segura do prontuário de medicamentos, impedindo o registro de dosagens inválidas e centralizando as instruções em um banco de dados local.

## 🛠️ Tecnologias Utilizadas
- **Python 3.12+**
- **Rich:** Para a criação da interface CLI moderna e colorida.
- **Pytest:** Para testes automatizados e garantia de qualidade.
- **Ruff:** Para linting e análise estática do código.
- **JSON:** Armazenamento persistente de dados.
- **GitHub Actions:** Pipeline de CI/CD para validação contínua.
